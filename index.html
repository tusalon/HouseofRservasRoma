Preguntas frecuentes

Respuestas rápidas antes de crear tu negocio.

Necesito instalar algo para usarlo?

No necesariamente. Puede funcionar como web instalable en Android y iOS. En planes premium también puede prepararse APK para Android.

Puede enviar recordatorios?

Sí. RservasRoma puede preparar avisos para turnos próximos, pagos pendientes y mensajes de seguimiento por WhatsApp.

Sirve para varios profesionales?

Sí. Cada servicio puede asignarse a profesionales específicos, con horarios y permisos separados.

Se puede cobrar anticipo?

Sí. El sistema puede dejar la cita pendiente hasta que la admin confirme el pago.

Se puede personalizar la imagen?

Sí. Logo, fondo, colores, texto de bienvenida, país, teléfono, redes y estilo visual del negocio.
