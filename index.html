Preguntas frecuentes

Respuestas rapidas antes de crear tu negocio.

Necesito instalar algo para usarlo?

No necesariamente. Puede funcionar como web instalable PWA. En planes premium tambien puede prepararse APK.

Sirve para varios profesionales?

Si. Cada servicio puede asignarse a profesionales especificos, con horarios y permisos separados.

Se puede cobrar anticipo?

Si. El sistema puede dejar la cita pendiente hasta que la admin confirme el pago.

Se puede personalizar la imagen?

Si. Logo, fondo, colores, texto de bienvenida, pais, telefono, redes y estilo visual del negocio.
